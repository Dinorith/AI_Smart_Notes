# AI Smart Note

AI Smart Note is an AI-powered application that helps users summarize documents, extract keywords, and generate flashcards efficiently.

## Features
- Upload PDF or text files for processing
- Generate AI-based summaries
- Extract important keywords
- Create interactive flashcards
- Responsive and user-friendly interface

## Technology Stack
- Frontend: React.js, Tailwind CSS, Vite
- Backend: Python or C++
- AI Model: OpenAI API
- Hosting: Vercel

## Installation
1. Clone the repository:
```bash
git clone https://github.com/Dinorith/AI_Smart_Notes.git
cd AI_Smart_Notes
```
2. Install dependencies:
```
npm install
```
3. Add .env with your API key:
```
OPENAI_API_KEY=your_api_key_here
PORT=5000
MODEL=gemini-2.5-flash
```
4. Run the project:
```
cd frontend
npm run dev

cd backend
node server.js
```
